# 21 Buenas Prácticas Adoptables (BPAs)

Las **BPAs** son intervenciones específicas, evidenciadas en IES de clase mundial, que la UDFJC puede adoptar para materializar la Reforma Vinculante. Cada BPA activa una o más de las **6 Retroalimentaciones Virtuosas (R1–R6)** del modelo QMT.
