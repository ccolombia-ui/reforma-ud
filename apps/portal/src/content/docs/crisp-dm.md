# Hoja de Ruta CRISP-DM — MI-12

**CRISP-DM** (Cross-Industry Standard Process for Data Mining) es el marco metodológico que organiza las 12 investigaciones del proyecto MI-12 en una secuencia coherente de comprensión → modelado → despliegue de la Reforma Vinculante.

## El ciclo aplicado a la Reforma

```
Business Understanding → Data Understanding → Data Preparation
         ↓                                              ↓
    Deployment    ←    Evaluation    ←      Modeling
```

## Por qué CRISP-DM

CRISP-DM impone disciplina narrativa: cada investigación tiene una pregunta trazadora, se apoya en datos e IES referentes reales, modela mecanismos, evalúa con indicadores y despliega con hoja de ruta. Evita el pensamiento desiderativo ("queremos ser como MIT") y ancla en evidencia accionable.
