## ¿Cómo contribuir?

Este portal es un Inner Source académico bajo TDD: cada paper, ficha BPA o guía de CoP nace como issue con criterios de aceptación, vive como Draft PR mientras se escribe, y solo se publica cuando supera la revisión.

- **Lee la guía**: [TDD para Comunidades de Práctica](https://github.com/ccolombia-ui/reforma-ud/blob/main/docs/governance/cop-tdd.md)
- **Abre un issue RED**: [GitHub Issues](https://github.com/ccolombia-ui/reforma-ud/issues/new/choose)
- **Pregunta al asistente**: usa el botón flotante 💬 abajo a la derecha
