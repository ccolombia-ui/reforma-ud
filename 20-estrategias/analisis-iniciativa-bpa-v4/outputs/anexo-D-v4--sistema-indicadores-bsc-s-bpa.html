Anexo D v4 · Sistema de Indicadores BSC-S Simplificado por BPA

Nota de versión: Este documento enriquece al Anexo D v3 (sistema de indicadores institucionales BSC-S) añadiendo el tablero de 20 KPIs transformacionales organizados por función misional y vinculados a las 21 BPAs. El Anexo D v3 conserva sus indicadores institucionales; el v4 añade la capa de medición por BPA. Es aditivo, no sustitutivo.

1 · Principios del BSC-S v4

1.1 · Diferencia entre indicador de actividad y de transformación

| Tipo | Ejemplo (mal) | Ejemplo (bien) | Por qué importa | |---|---|---|---| | Actividad | "Número de cursos dictados" | "% cursos con metodología PBL validada" | Contar cursos no mide si cambió la formación. | | Actividad | "Número de proyectos de investigación" | "% proyectos con participación undergrad" | Contar proyectos no mide si los estudiantes investigan. | | Actividad | "Número de convenios de extensión" | "% convenios con métrica ODS publicada" | Contar convenios no mide impacto territorial. | | Transformación | "Estudiantes satisfechos con la carrera" | "Índice de integración misional (promedio R1–R6)" | La satisfacción es subjetiva; la integración es sistémica. |

1.2 · Reglas del BSC-S v4

  1. 20 KPIs fijos: 4 funciones × 5 criterios. No se añaden ni quitan sin reforma del anexo.
  2. Línea base obligatoria: todo KPI tiene AS-IS 2026. Si no se sabe, se declara "N/A (estimar en Año 0)".
  3. BPA activador único: cada KPI se vincula a 1–2 BPAs. Si el KPI no mejora, se revisa la BPA.
  4. Confianza epistémica: valor 0–1. < 0,6 = estimado; ≥ 0,6 = medido.
  5. Frecuencia: C1–C3 semestral; C4–C5 anual.

7.2 · Dashboard DataviewJS (consulta ejemplo)

dataviewjs
// Tablero F4 · Integración
const kpis = dv.pages()
  .where(p => p.kpi_id && p.kpi_id.startsWith("F4"))
  .sort(p => p.criterio);

dv.table(
  ["KPI", "Valor", "AS-IS", "TO-BE", "% Avance", "Semáforo"],
  kpis.map(k => {
    const avance = ((k.valor - k.as_is) / (k.to_be - k.as_is) * 100).toFixed(1);
    const color = avance >= 80 ? "🟢" : avance >= 50 ? "🟡" : "🔴";
    return [k.kpi_nombre, k.valor + "%", k.as_is + "%", k.to_be + "%", avance + "%", color];
  })
);

7.3 · Integración con Anexo D v3

Los 20 KPIs v4 se suman a los indicadores institucionales del Anexo D v3. La DGE&P consolida ambos en un informe único:
  • Indicadores v3: permanencia institucional, eficiencia terminal, cobertura, etc. (se mantienen).
  • Indicadores v4: 20 KPIs transformacionales por BPA (se añaden).
La frecuencia de publicación del informe consolidado: semestral.

8 · Glosario de indicadores v4

| Término | Definición | |---|---| | AS-IS | Valor medido o estimado en la línea base (2026). | | TO-BE | Meta proyectada para 2034. | | BPA activador | Buena Práctica Adoptable responsable directa del movimiento del KPI. | | Confianza epistémica | Grado de certeza en la línea base (0 = especulación; 1 = medición directa). | | Índice de integración misional | Promedio de intensidad de las 6 retroalimentaciones QMT (R1–R6). | | Semáforo | 🟢 ≥ 80 % avance; 🟡 50–79 %; 🔴 < 50 % o tendencia negativa. | | Transformación | Cambio medible en comportamientos institucionales, no solo en actividades realizadas. |
